# Password Generator App (Console)

This is a simple Python-based Password Generator App that runs in the console.

## Features

- Customizable password length
- Option to include uppercase letters, digits, and special characters
- Secure password generation using `random` module

## Steps to Run in PyCharm IDE

1. Open PyCharm IDE.
2. Click `File` > `Open` and select the `PasswordGeneratorApp` directory.
3. Run `password_generator.py` to start the application.
4. To run the unit tests, right-click on `test_password_generator.py` and select `Run`.

## Test Cases

- Check default password length (12 characters)
- Validate custom length
- Ensure character set inclusion (uppercase, digits, specials)
- Handle case where no character sets are selected
